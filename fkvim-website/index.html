Get Started

Installation

Requirements

  • Neovim >= 0.9.0
  • Git
  • Nerd Font (Recommended)
  • Curl/Wget (Optional)

Installation Steps

git clone https://github.com/TheFlashCodes/FKvim.git ~/.config/nvim
nvim

On the first launch, Fkvim will automatically install all the required plugins using lazy.nvim.

Setups

Fkvim is designed to be modular and configurable. The main configuration is handled in init.lua, which loads different modules for plugins, settings, and keybindings.

File Structure

The configuration is organized into several directories inside lua/:

  • fk_configs/: Core configurations for Neovim.
  • fk_plugins/: Plugin specifications and configurations.
  • fk_custom/: Your custom configurations that are not tracked by git.

Plugin Management

Fkvim uses lazy.nvim for plugin management. You can add or remove plugins in the files under lua/fk_plugins/.

Configuration

You can customize Fkvim by modifying the files in the lua/ directory. For personal customizations that you don't want to commit to your fork, you can use the lua/fk_custom/ directory.

Keybindings

Keybindings are defined in lua/fk_configs/fk_binds.lua. You can change existing keybindings or add new ones in this file.

Adding New Plugins

To add a new plugin, you can create a new file in lua/fk_plugins/ and add the plugin specification there. For example, to add a new plugin my-plugin.lua:

return {
    'user/my-plugin',
    -- lazy.nvim options
}
